# midnight.nvim
A dark, muted colorscheme for Neovim.

<div align="center">
  <img src="static/preview.jpg" alt="preview" width="900">
</div>

## Installation 
```lua
vim.pack.add({
  "https://github.com/sudoscrawl/midnight.nvim"
})

vim.cmd.colorscheme("midnight")
```
